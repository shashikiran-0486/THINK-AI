THINKBOLD — Deployment Guide

This project is a Vite + React + TanStack Start + Supabase app. Follow the steps below to deploy it.

1 · Install dependencies

npm install

2 · Configure environment variables

Copy .env and fill in your Supabase credentials:

VITE_SUPABASE_URL=https://YOUR_PROJECT.supabase.co
VITE_SUPABASE_PUBLISHABLE_KEY=YOUR_ANON_KEY
VITE_SUPABASE_PROJECT_ID=YOUR_PROJECT_ID

3 · Run Supabase migrations (adds all 24 products)

npx supabase db push
# or link and push:
npx supabase link --project-ref YOUR_PROJECT_ID
npx supabase db push

4 · Deploy the Edge Function (ThinkAi chatbot)

npx supabase functions deploy thinkai

Set the LOVABLE_API_KEY secret in your Supabase dashboard → Edge Functions → Secrets.

5 · Run locally

npm run dev

6 · Build for production

npm run build

Output goes to dist/. Deploy to Cloudflare Pages, Vercel, Netlify, or any static host.
